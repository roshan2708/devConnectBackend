# FRONTEND MASTER PROMPT

## Overview
This document serves as the master prompt for frontend development, detailing guidelines, best practices, and coding standards.

## Coding Standards
- Follow the latest JavaScript (ES6+) and CSS3 standards.
- Use consistent naming conventions for variables and functions.
- Ensure all code is well-commented for clarity.

## Frameworks and Tools
- Utilize React.js for building user interfaces.
- Employ Redux for state management.
- Use Axios for making HTTP requests.

## Styling
- Stick to CSS-in-JS for styling components.
- Use styled-components for styling React components.

## Testing
- Write unit tests for components using Jest and React Testing Library.
- Ensure all tests are passing before merging changes.

## Deployment
- Use GitHub Actions for CI/CD pipelines to automate deployment.
- Ensure the main branch is always in a deployable state.
